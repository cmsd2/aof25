# aof25 :: Advent of Fortran 2025

Fortran code solution(s) to aoc2025 [https://adventofcode.com/2025](https://adventofcode.com/2025).

## Build

Requirements:
 1. fpm [https://fpm.fortran-lang.org/](https://fpm.fortran-lang.org/)

```
fpm build
```
## Running

Requirements:
 1. Advent of code data files at `data/day01/input.txt` etc

```
fpm run day1
```
